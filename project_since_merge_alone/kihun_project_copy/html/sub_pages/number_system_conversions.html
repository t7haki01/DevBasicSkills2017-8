    • From To
    In this page, math tool Number system conversions is able for converting
    from one numbering system to the other numbering system.
    In here, prefix is used for representing each numbering system.
    (Decimal = " ", Binary = "0B", Octal = "0O", Hexadecimal = "0X")
    So if you remove then it won't work, you can always write down again prefix

    < Instruction >
    1.Select which numbering system you want to change from next to "From"
    2.Write down number in "Input box"
    3.Then choose which numbering system you want to get from next to "To"
    4.Then press "Convert"

    ⚠ In input box, each numbering systems have restricted input
    You can put only Decimal(number 0 ~ 9), Binary(number 0, 1, letter "B"),
    Octal(number 0 ~ 7, letter "O"), Hexadecimal(number 0 ~ 9, letter a,b,c,d,e,f,x)

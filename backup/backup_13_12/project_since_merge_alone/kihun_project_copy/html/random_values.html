    • This is Project Work web based mathematics tool.
    The tools offer six different math functionalities for the user.
    Firstly, there is Number system conversions for converting from one numbering system to the other numbering system.
    Secondly, Number system outputs for showing decimals 0-50 in binary, octal, and hexadecimal.
    Thirdly, Combinatorics for calculating combinations or permutations,
    in case of permutations providing sampling without replacement or with replacement.
    Forthly, Truth tables for checking basic set of truth tables for basic operations.
    Fifthly, Random values for creating number of random numbers in different decimal.
    Lastly, Trigonometric function values for checking Sin, Cos, Tan degrees in numberic values.
